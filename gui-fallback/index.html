⚠ Restore backup

Restoring:

This will replace the live DB on this node only. Other nodes are NOT updated. Their gen is higher, so they will push their current state back to this node on the next sync, overwriting this restore. Use Force Restore to propagate to peers.
⚠ FORCE RESTORE — this node's gen will be set above all peer gens. On the next sync drain, this node will push the restored state to ALL peers, overwriting their current data. Only use this for disaster recovery or corruption fixes.
